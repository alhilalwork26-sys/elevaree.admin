🔥 Konfigurasi Firebase

Ikuti langkah berikut agar data tersimpan online dan bisa diakses dari mana saja

1

Buat Project Firebase

Buka console.firebase.google.com → Klik "Add Project" → Beri nama (misal: elevare-academy) → Klik Continue → Create Project.

2

Aktifkan Firestore Database

Di sidebar kiri → klik Firestore Database → "Create database" → pilih Start in test mode → pilih region → Enable.

3

Aktifkan Authentication

Di sidebar kiri → klik Authentication → "Get started" → pilih Email/Password → Enable → Save.

Lalu klik tab Users → "Add user" → masukkan email dan password admin Anda.

4

Salin Config ke File HTML

Di Firebase Console → klik ikon ⚙️ Project Settings → scroll ke bawah → klik icon Web (</>) → copy firebaseConfig.

Buka file HTML ini dengan Notepad/VS Code → cari bagian FIREBASE CONFIG → ganti nilai YOUR_... dengan milik Anda:

const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "nama-project.firebaseapp.com",
  projectId: "nama-project",
  storageBucket: "nama-project.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123..."
};
5

Hosting (Opsional)

Upload file HTML ke layanan hosting seperti Firebase Hosting, Netlify, atau Vercel — semuanya gratis untuk project kecil. Cukup drag & drop file HTML-nya.
